Introduction

Welcome to Text Launcher X (hereinafter referred to as "this application"). We highly value your privacy protection, and this privacy policy aims to explain how this application handles your information.

Information Collection

This application collects the following information during operation:

  • Application list information: To display and manage applications, this application will obtain the list of installed applications on your device, including application names and package names.
  • Basic device information: This application may obtain basic device information, such as device model, Android version, etc., to ensure the application runs normally.

Information Usage

The information collected by this application is only used for the following purposes:

  • Displaying and managing application lists
  • Providing application search functionality
  • Saving desktop application shortcut settings
  • Ensuring the application runs normally on your device

Network Connection

This application does not connect to the Internet, and all functions are implemented locally.

Data Storage

This application uses SharedPreferences to store user settings, such as desktop application lists. This data is only stored locally on your device and will not be uploaded to any server.

Information Sharing

This application does not share your information with any third parties, nor does it record user usage behavior.

Permission Description

This application requires the following permissions:

  • RECEIVE_BOOT_COMPLETED: Allows the application to start when the device boots, in order to update time and date display in a timely manner.
  • QUERY_ALL_PACKAGES: Allows the application to obtain information about all installed applications on the device, in order to display and manage application lists.

Security Measures

We have taken the following security measures to protect your information:

  • All data is only stored locally on the device
  • No personal identity information is collected
  • No connection to any external servers
  • No user usage behavior is recorded

Privacy Policy Updates

We may update this privacy policy from time to time. When we update the privacy policy, we will notify you within the application.

Contact Information

  • Email: itoolstudio@icloud.com

Scope of Application

This privacy policy only applies to this application and does not apply to other applications or websites accessed through links in this application.
